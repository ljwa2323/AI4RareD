## 核心主题速览

### 诊断技术路线

- **表型驱动**：HPO 术语化 -> 疾病相似度检索（如 Phenomizer、Exomiser）
- **知识增强 LLM**：RAG + 知识图谱（Orphanet / OMIM / HPO）
- **多智能体编排**：表型抽取、疾病归一化、病例匹配、基因分析等专用代理协作（如 DeepRare、RDguru）

### 关键数据资源

| 资源 | 主要用途 |
|---|---|
| [OMIM](https://www.omim.org/) | 基因-表型孟德尔遗传知识 |
| [Orphanet](https://www.orpha.net/) | 罕见病标准命名、ORPHA 编码与多维资源 |
| [DECIPHER](https://www.deciphergenomics.org/) | 患者级基因型-表型共享与匹配 |
| [HPO](https://hpo.jax.org/) | 人类表型本体 |
| [UMLS / SNOMED / ICD](./医学编码体系与互联资源全景梳理.md) | 跨体系编码映射与临床互操作 |
